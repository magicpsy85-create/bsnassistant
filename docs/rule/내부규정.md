# 계약성과인정 규정
1. 원칙적으로 단독 빌딩 매매 계약 성사시 매도 또는 매수 각각 1건으로 인정. 도급중개사는 총5건의 성과 충족 시 파트너중개사의 자격을 얻음. 단, 5건 중 매수 최소 1건이 포함되어야하며 이는 밑에 예외적인 경우도 동일하게 적용함.
2. 예외적으로 단독 빌딩의 매매가 아닌 그외 부동산 중개의 경우 매도, 매수,임대, 임차 각각 1건으로 보았을때 중개수수료가 3,000만원 이상 발생하면 빌딩 중개 1건과 같음.
3. 예외적인 경우에서 1건의 중개수수료가 3,000만원 미만인 경우 해당 금액은 3,000만원까지 누적이 가능하며 누적 달성이 되는 경우 성과 1건으로 인정되며 누적 금액은 초기화됨.
4. 예시 첫번째. 누적 2,000만원인 상태에서 하나의 중개 계약 중 매도,매수,임대,임차측 중 하나를 담당하여 1,000만원 이상의 수수료가 발생되면 빌딩 중개 성과 1건 인정. 누적금액은 0원으로 초기화.
5. 예시 두번째. 누적 2,000만원인 상태에서 하나의 중개 계약 중 매도와 매수측 또는 임대와 임차측 양쪽을 담당(양타)하여 한쪽에서 1,000만원과 다른 한쪽에서 3,000만원의 수수료가 발생되면 기존 누적 수수료와 한쪽 수수료 합산으로 성과 1건 인정. 다른 한쪽 수수료로 성과 1건 인정. 총 2건으로 인정됨.

# CMS 규정
1. 매도의 경우 등기부상 소유자의 이름, 연락처, 생년월일, 주소 필수 기재. 매수의 경우 매수자의 이름, 연락처 필수 기재. 예외로 매도나 매수의 실질적인 위임을 받았다고 판단할 사정이 있는 경우는 그 대리인의 정보 입력을 인정함.
2. 연락처 누락인 경우 연락처를 취득하여 입력한 사람을 담당자로 변경함.
3. 외부업체 전속인 경우 등기부상 소유자의 연락처 없이 전속부동산의 번호만 있으면 소유자의 연락처를 취득하여 입력한 사람을 담당자로 변경함.
4. 매물정보 중 임대차내역 제외한 빈 항목 없이 모두 채워 넣고 작업내역 입력을 완료해야 진행가능 상태로 인정함.
5. 매물정보 중 각 임차인별 세부임대정보가 아닌 총합계로 입력된 경우 진행가능 변경일로부터 7일 이내 세부임대정보를 입력해야하며 미입력 시 진행불가로 상태 변경함.
6. 계약 사실을 알게 된 매물은 빌딩명 끝부분에 '@'를 입력하고 잔금 확인 후 '@'를 삭제 후 진행불가 상태로 변경함.
7. 진행가능으로 상태 변경 후 그 매물을 빌딩목록 화면에서 체크박스로 선택하여 최상위로 버튼을 클릭하면 CMS 빌딩목록 초기화면 상단에 보여지기 때문에 다른 중개사들에게 더 많은 노출이 되어 계약 성사 확률이 높아짐.
8. 자료버튼에는 등기부, 토지대장, 건축물대장, 도면 등의 브리핑에 참고할 공적장부 및 자료를 업로드함.
9. 매물상태 중 진행가능은 현 시점 당장 매매가 가능한 상태를 뜻하며 진행불가는 현 시점 당장 매매가 불가한 상태를 뜻함.
10. 인쇄모드의 경우 최적화를 위한 설정이 필요함. 크롬 브라우저 사용을 권장하며 인쇄 설정창에서 레이아웃은 가로방향, 용지크기는 A4, 여백은 없음, 배율은 133%, 옵션은 배경 그래픽 선택과 머리말 선택 해제로 설정함.
11. 도급계약서에 따라 도급비용을 지급함.


# 담당자 변경 규정 공통
1. 변경 무효를 주장하는 인원은 그 증명을 할 수 있는 증거자료를 변경 요청 당일 이내 상대 파트너중개사에 제출하여야 함.
2. 규정 이외의 담당자 변경에 관한 다툼이 있을 경우 당사자 합의가 우선 원칙이며 합의가 어려울 경우 파트너 회의 및 투표를 진행하여 결정함.

# 담당자 변경 규정 매도
1. 접수는 시간제한 없이 카카오톡 플러스친구 'BSN빌사남부동산중개법인'으로 접수(건물명, 주소, 고객정보, 기존담당자, 신청인, 사유).
2. 변경은 매주 월~금 09:30 부터 17:30 까지 법인에 접수 된 순서대로 처리함.
3. CMS에 최초 신규 입력 된 그 시점부터 14일 이내 담당자 변경 불가.
4. CMS에 최초 신규 입력 상태는 진행불가로 정함.
5. 진행불가에서 진행가능으로 상태변경 또는 담당자가 변경된 경우 그 시점부터 14일 이내는 담당자 변경 불가(규정에 따른 변경 무효 제외).
6. 진행가능에서 진행불가로 변경 된 매물은 원칙적으로 그 시점부터 14일 이내 담당자 변경 불가함. 단, 진행가능 매물이 최근 작업 내역 날짜 기준 60일 이내 다음 작업 내역 없을 시 진행불가 및 담당자 없음 상태로 자동 변경된 경우 통화 또는 미팅 작업 내역을 입력 하여 담당자 변경 및 진행가능 상태로 변경 요청 가능.
7. 진행가능 매물가격(최저) 기준 매매가의 5% 이상 네고 시 원본 녹취 파일(편집본 제출 시 반려) 제출 후 담당자 변경 요청 가능.
8. 녹취 내용에는 물건 지 주소, 소유주 성명(대리인일 경우 관계 확인), 매매 금액 필수 포함.
9. 매도인이 법인일 경우 원칙적으로 대표자 컨택이 우선이며 예외적으로 법인관계자랑 연락하는 경우 정확한 이름, 직책, 휴대폰 연락처 파악 필요.
10. 외부업체 전속 매물의 경우 그 업체를 통한 네고는 위 녹취 내용에 추가로 소유주에게 전속중개에 관한 위임을 받았다는 업체의 진술 내용 포함하여 제출 후 담당자 변경 요청 가능.
11. 외부업체 전속 매물의 경우 소유주가 직접 전속의 무효를 언급하는 원본 녹취 파일을 제출하거나 담당중개사가 직접 전속 계약을 받아오는 경우 담당자 변경 가능.
12. 매물 작업 내역에 '작업중'을 남긴 날짜로 부터 작업 독점기간은 7일로 정하며 동일한 팀에서 중복 입력은 인정되지 않음.
13. 매도인 요청으로 담당자 변경 불가하며 해당하는 파트너중개사 협의가 원칙임. 5% 네고 규정과 전속중개계약을 2개월 이상 받아 오는 경우에만 협의 없이 변경 신청만으로 가능(전속중개계약서 중개법인 우선 날인 필수).
14. 파트너중개사 계약해지 시 매물은 즉시 중개법인으로 귀속 최대 60일 이후 위 규정 동일 적용.
15. 도급중개사 계약해지 시 매물은 즉시 해당 파트너중개사에게 귀속.
16. 외부 전속 매물의 경우 진행불가 원칙이나 불가피한 경우 매가의 0.3%이상 수수료 발생 시 빌딩 중개 계약 인정(매가의 0.3%미만 수수료 발생 시 빌딩 중개 계약 미인정 및 수수료 정산 매수8:매도2 비율 적용) – 일반 계약의 경우에도 적용, 임대의 경우 0.3% 미만의 수수료 발생 시 누적에도 불포함.

# 담당자 변경 규정 매수
1. 매수인이 중개법인에 직접 전화하여 담당자변경을 요청하는 경우 변경 가능.
2. 직전 작업 내역 날짜 기준 30일 이후 작업 내역 없을 시 보류 그룹 자동 변경됨.
3. 보류로 자동 변경된 날부터 누구든 작업 가능(통화, 미팅 작업 내역 입력 필수).
4. 파트너중개사 계약해지 시 매수 고객은 즉시 중개법인으로 귀속되며 60일 이후 위 규정 동일 적용(위탁 고객 상태 유지).
5. 도급중개사 계약해지 시 매수 고객은 즉시 소속 파트너중개사에게 귀속.
6. 파트너중개사를 통해 계약 한 매수 고객(회사 위탁 제외)의 계약 매물(고객 소유의 다른 매물이 회사 계약이 아닌 경우 다른 인원 작업 가능)은 해당 파트너십 계약 기간 중 영구 귀속됨. 단, 고객의 요청이 있을 경우 예외.
